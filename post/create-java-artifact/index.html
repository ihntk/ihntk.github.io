Cтворення артифакта Java

 2020-08-20  168 слів  java dev 

Заходимо в Project structure > Artifacts. Якщо є якийсь артифакт, видаляємо його(-) і натикаємо + > JAR > From modules with dependencies, вибираємо головний клас. Якщо до цього був якийсь маніфест, бажано його видалити, щоб ідея не лаялась > OK

Меню Build > Build Artifacts… Вибираємо створений артифакт. Забираємо файл jar в каталозі out

Таким чином можна зібрати програму з усіма бібліотеками, включаючи javaFX, інтерфейс якої можна запустити з консолі через прапорець -g. Напряму з вінди з 14 джавою і javaFX зібрати javaFX артефакт не вдалося, так само як і за допомогою maven

Останні спостереження

В Project structure > Project вибрана SDK 11(zulu) - впливає на те яку jvm використовуватиме maven - на 18 не хотів збирати > Libraries натискаємо + додаємо java бібліотеку - вказуємо шлях де розміщена javafx (/lib/jvm/java-19-openjdk/lib) - після цього зникають всі помилки повʼязані з тим, що компілятор на розуміє бібліотек javafx > Platform Settins > SDKs є 11 і 19, вибрана 19 (не знаю як потрібно правильно) В Run/Debug Configuration вибрана 11 sdk
