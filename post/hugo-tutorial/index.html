Коротка інструкція з Hugo

 2020-08-20  138 слів  website 

Створення сайта

hugo new site (path/to/)siteName

створює сайт (у вказаному шляху). Потрібно скопіювати тему в каталог themes/ і додати в файл конфіга:

echo 'theme = "themeName"' >> config.toml

Щоб створити сторінку:

hugo new post/my-post.md

створює сторінку в каталозі content/post. В title сторінки підставляє назву файла, замінюючи знаки - на пробіли

Збірка і публікація

hugo

команда збирає сайт і розміщує його в каталозі public/

флаги:

-D	--buildDrafts	(draft: true)
-E	--buildExpired	(expirydate)
-F	--buildFuture	(publishdate)

перед кожною збіркою потрібно видалими public/ , інакше це може призвести до помилок

Розробка

hugo server

запускає сервер http://localhost:1313/, який в реальному часі оновлює всі зміни, що відбуваються в наступних місцях:

static/*
content/*
data/*
i18n/*
layouts/*
themes/<CURRENT-THEME>/*
config

щоб запустити сервер без оновлення в реальному часі використувуюють наступні команди:

hugo server --watch=false

або

hugo server --disableLiveReload

або вказати в файлі конфіга:

disableLiveReload = true
