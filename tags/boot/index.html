Установка в UEFI

Потрібно установити пакети grub і efibootmgr.
Якщо OS завантажена в режимі UEFI, системний завантажувальний розділ EFI змонтований в /boot/efi, то для установки завантажувача потрібно виконати команду

# grub-install
